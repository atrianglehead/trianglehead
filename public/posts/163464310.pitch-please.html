The Gear You’ll Need

Here are 12 pitches we’ll be needing in this part in order of increasing pitch: Ma_, Pa_, Dha_, Ni_, Sa, Re, Ga, Ma, Pa, Dha, Ni and Sa'.

Each of the 5 melodies uses only a subset of the above pitches.

Since having a single pitch dictionary with all 12 required pitches might be confusing for a lot of people, each melody will use its own separate, focused pitch dictionary.

To make it easier to recognise each melody’s pitches, we’ll add a Sa (an octave below, so technically Sa_) simultaneously with every pitch of every melody , matching the loudness of each original pitch.

We’ll also use a slower tempo and use the same Sa for all melodies (even though the original melodies do not share the same Sa).

Coupled with the modified version of each melody, this will be all you’ll need to name each pitch.

Make sure to also listen to the simplified melodies by heading over to Part 26. This will help you in contextualising the modified versions that you’ll find below.
